Social → interaction/relationships: initiating/joining peers, group participation, social cues, perspective taking, roles/boundaries, leisure with peers, compliments, fair play.

Tie-breakers

Mentions peers/group/join/play/compliment/discussion → likely Social.

Mentions ask/tell/say/respond/question/answer/retell/follow (verbal) directions/sentence/voice → likely Communication.

Mentions comply/refuse/persist/wait/cope/attention/transition/organize/hygiene → likely Behavior.

Eye Contact: if framed as language mechanic → Communication; if framed as social reciprocity/cues → Social.

Honesty: if framed as discuss/express → Communication; if framed as reducing lying → Behavior (Maladaptive).

Goal-Level Decision Rules (flexible, conceptual)

LTG (umbrella): broad capability or theme (e.g., Compliance, Coping/Emotion Regulation, Attention, Conversation Skills, Manding, Perspective Taking, Group Skills, Task Completion, Responsibility for Belongings).

Accept synonyms (e.g., Coping Skills ≈ Emotional Regulation; Conversation Skills ≈ Pragmatic Language; Self-Advocacy ≈ Functional Communication).

STG (facet): specific sub-skill of an umbrella (e.g., Selective Attention, Staying on Topic, Group Instruction Follow-Through, Manding for a Break, Initiating games with peers, Using Proper Tone and Volume, Organizing personal items).

Target — Out of Scope: includes metrics or programming elements (any of: time/duration, counts/frequencies, %, prompts/latency, “Once per Session,” “across X sessions,” “Criteria/Mastery,” or program components like MO/SD/Prompting/Error Correction/Reinforcement).
